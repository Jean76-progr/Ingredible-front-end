Installation
Prérequis
Python 3.11 (assurez-vous que Python est installé sur votre machine)
Étapes d'installation
Cloner le projet depuis GitHub :

bash
Copy code
git clone https://github.com/votre_utilisateur/nom_du_projet.git
Accéder au répertoire du projet :

bash
Copy code
cd nom_du_projet
Créer un environnement virtuel :

bash
Copy code
python -m venv venv
Si vous utilisez une version spécifique de Python (par exemple Python 3.8), remplacez python par python3.8.

Activer l'environnement virtuel :

Sur Windows (PowerShell) :

bash
Copy code
.\venv\Scripts\Activate
Sur Linux/macOS :

bash
Copy code
source venv/bin/activate
Installer les dépendances :

bash
Copy code
pip install -r requirements.txt
Lancer l'application :

bash
Copy code
python app.py
L'application sera accessible à l'adresse http://localhost:5000 dans votre navigateur.

Utilisation
Créer un compte :

Accéder à http://localhost:5000/create_account.
Remplir le formulaire et valider.
Se connecter :

Accéder à http://localhost:5000/login.
Utiliser le nom d'utilisateur et le mot de passe créés précédemment.
Autres fonctionnalités :

Explorer les différentes fonctionnalités de l'application, notamment la création de recettes, la gestion des ingrédients, etc.
